# 📸 SnapIt – A Social Platform for College Students  

SnapIt is a social platform designed for college students to share images of events and social clubs, similar to Pinterest. The platform enables users to post, discover, and engage with campus activities in real-time.  

## deployedLink : https://snapit1.netlify.app/
## 🚀 Features  
✅ **User Authentication** – Secure login and registration system  
✅ **Image Upload** – Easily upload and share event images  
✅ **State Management** – Efficient data handling with Redux  
✅ **Real-Time Updates** – Instant event notifications and interactions  
✅ **Modern UI** – Responsive and sleek design with Tailwind CSS  

## 🛠️ Tech Stack  
- **Frontend:** React, Redux, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Other:** Cloudinary (for image storage), JWT (authentication)
